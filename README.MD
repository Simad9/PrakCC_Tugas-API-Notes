# API Notes

### Description

Ini adalah Tugas Prak CC. Disuruh buat API Notes yang mengimplementasikan CRUD

### How to Copy

1. Clone Repository

```bash
git clone https://github.com/Simad9/PrakCC_Tugas-API-Notes
```

2. Install Dependencies

```bash
npm install
```

3. Run Server (gunakan command dibawah untuk menjalankan server)

```bash
npm run dev
```

4. Silahkan buka aplikasi untuk mengecek API nya

### API Documentation
Mengambil data notes \
``` GET /notes ```

Mengambil data notes berdasarkan id nya \
```GET /notes/:id```

Membuat data notes \
```POST /notes```

Mengubah data notes \
```PUT /notes/:id```

Menghapus data notes \
```DELETE /notes/:id```
